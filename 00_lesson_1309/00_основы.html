Первый вариант списка - <ul>:

  • Первый пункт
  • Второй пункт

Второй вариант списка - <ol>:

  1. Покормить кота
  2. Погладить кота
  3. Получить трындюлей от кота
  4. Положить кота спать

У списков можно менять начало нумерации при помощи атрибута start (только для <ol>)

  1. первый
  2. второй

У списков можно менять стиль пунктов при помощи атрибута type:

  1. i - маленькие римские цифры (iv, v, viii);
  2. I - большие римские цифры (I, II, IV);
  3. a - маленькие буквы (a, b, c...);
  4. A - большие буквы (A, B, C...);

В элементы списков можно вкладывать тег ссылки - <a> - anchor - ссылка.

У тега ссылки есть обязательный атрибут href - в нем указывается ссылка, по которой нужно перейти пользователю.
