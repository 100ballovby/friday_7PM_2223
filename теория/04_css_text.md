<section class="course-theory__content"><h1 class="course-theory__content-heading course-theory__content-heading--synopsis">Конспект «Оформление текста»</h1><div class="course-theory__content-text"><h2>Свойство font-size</h2><p>Управляет размером шрифта. Значение свойства задаёт желаемую высоту символа шрифта. Причём единицы измерения могут быть абсолютными или относительными.</p>

<section class="course-theory__content"><div class="js--theory-pane__theory course-theory__content-inner"><h1 class="course-theory__content-heading">Типы значений: абсолютные и&nbsp;относительные</h1><div class="course-theory__content-text"><p>Давайте сделаем аватарку модной, то&nbsp;есть круглой. Можно увеличить значение <code>border-radius</code> до&nbsp;половины размера аватарки, то&nbsp;есть до&nbsp;<code>40px</code>, и&nbsp;получить круг. Но&nbsp;если потом появится новая аватарка с&nbsp;другими размерами, то&nbsp;снова придётся менять стили. Можно&nbsp;ли сразу сделать универсально? Да! С&nbsp;помощью процентов:</p><pre><code>border-radius: 50%;</code></pre><p>Если радиус скругления задать в&nbsp;процентах, то&nbsp;он&nbsp;будет рассчитываться <em>относительно</em> размеров элемента и&nbsp;будет изменяться вместе с&nbsp;этими размерами. Если этот радиус задать в&nbsp;пикселях, то&nbsp;он&nbsp;<em>абсолютно</em> всегда будет одинаковым. Вот и&nbsp;единицы измерения в&nbsp;CSS делятся на&nbsp;<i>абсолютные</i> и&nbsp;<i>относительные</i>.</p><p>Абсолютные единицы измерения привязаны к&nbsp;настоящим физическим размерам и&nbsp;связаны между собой жёсткими пропорциями. Примеры абсолютных единиц измерения:</p><pre><code>font-size: 1cm;  /* 1 сантиметр  */
font-size: 10mm; /* В 1 сантиметре 10 миллиметров */
font-size: 38px; /* В 1 сантиметре 38 <del>попугаев</del> пикселей */
</code></pre><p>Пиксели, <code>px</code>, используют чаще всего, остальные абсолютные единицы почти не&nbsp;применяют.</p><p>Относительные единицы измерения описывают значения, которые зависят от&nbsp;других значений. Например, ширина элемента в&nbsp;процентах зависит от&nbsp;ширины родительского элемента, а&nbsp;ширина элемента в&nbsp;<code>em</code> зависит от&nbsp;размера шрифта самого элемента.</p><p>К&nbsp;относительным единицам относятся <code>em</code>, <code>rem</code>, <code>vh</code>, <code>vw</code> и&nbsp;некоторые другие, ну&nbsp;и,&nbsp;конечно&nbsp;же, проценты. Каждая из&nbsp;таких единиц решает свой круг задач. Например, проценты используют для «резиновой» вёрстки, а&nbsp;<code>em</code> применяют в&nbsp;вёрстке государственных сайтов с&nbsp;особыми дополнительными требованиями к&nbsp;масштабированию текста.</p><p class="info info--frame-heading info--frame-heading--boremode info--gray">
Перечень всех <a href="https://www.w3.org/TR/css3-values/#absolute-lengths" rel="nofollow noopener" target="_blank">абсолютных единиц измерения</a> и&nbsp;их&nbsp;соотношений есть в&nbsp;спецификации. Там&nbsp;же, в&nbsp;спецификации, есть перечень всех <a href="https://www.w3.org/TR/css3-values/#relative-lengths" rel="nofollow noopener" target="_blank">относительных единиц измерения</a> и&nbsp;описание правил расчёта.</p></div></div></section>

<p>Самая часто используемая единица измерения размера шрифта&nbsp;— пиксели <code>px</code>:</p><pre><code>p {
  font-size: 20px;
}</code></pre><p>Но, чтобы при изменении основного размера шрифта родителя его дочерние элементы пропорционально меняли свои размеры шрифта, есть специальная единица измерения&nbsp;— <code>em</code>.</p><p>Величина <code>1em</code>&nbsp;— это <i>такой же</i> размер шрифта, что и&nbsp;у&nbsp;родителя. Соответственно, если нужно, чтобы шрифт дочернего элемента был всегда в&nbsp;2 раза больше родительского, то&nbsp;надо задать значение <code>2em</code>:</p><pre><code>h1 {
  font-size: 2em;
}</code></pre><h2>Свойство line-height</h2><p>Свойство управляет высотой строки или межстрочным интервалом.</p><p>По&nbsp;умолчанию это свойство имеет значение <code>normal</code>. Оно указывает браузеру, что межстрочный интервал нужно подобрать автоматически, исходя из&nbsp;размера шрифта. Спецификация <a href="https://www.w3.org/TR/CSS2/visudet.html#propdef-line-height" target="_blank" rel="noopener nofollow"></a>рекомендует устанавливать его в&nbsp;пределах <code>100-120%</code> от&nbsp;размера шрифта. То&nbsp;есть:</p><pre><code>p {
  font-size: 10px;
  line-height: normal; /* значение будет примерно 12px */
}</code></pre><p>Значение <code>normal</code> позволяет всем нестилизованным текстам выглядеть удобочитаемо. Однако, если есть необходимость отойти от&nbsp;стилизации по&nbsp;умолчанию, <code>line-height</code> можно задать фиксированное абсолютное значение в&nbsp;<code>px</code>.</p><pre><code>p {
  font-size: 16px;
  line-height: 26px;
}</code></pre><p>Если нужно задать <code>line-height</code> относительное значение, но&nbsp;не&nbsp;такое, как <code>normal</code>, то&nbsp;значение задаётся в&nbsp;процентах или в&nbsp;виде множителя. В&nbsp;таком случае браузер вычисляет значение динамически в&nbsp;зависимости от&nbsp;<code>font-size</code>:</p><pre><code>p {
  font-size: 10px;
  line-height: 150%; /* вычисленное значение: 10px * 150% = 15px */
  line-height: 2;    /* вычисленное значение: 10px * 2 = 20px */
}</code></pre><p>Относительные значения более гибкие, чем абсолютные. Но&nbsp;для простых сайтов фиксированных <code>font-size</code> и&nbsp;<code>line-height</code> будет вполне достаточно.</p><h2>Свойство font-family</h2><p>В&nbsp;значении этого свойства указывается список названий шрифтов, перечисленных через запятую. В&nbsp;начале списка располагают самый желаемый шрифт, затем менее желаемый, а&nbsp;в&nbsp;самом конце списка&nbsp;— общий тип шрифта. Браузер проходит по&nbsp;списку слева направо и&nbsp;использует первый найденный в&nbsp;системе или на&nbsp;сайте шрифт. Если название шрифта состоит из&nbsp;нескольких слов, то&nbsp;его нужно заключать в&nbsp;кавычки.</p><pre><code>body {
  font-family: "PT Sans", "Arial", sans-serif;
}</code></pre><h2>Свойство font-weight</h2><p>Свойство задаёт насыщенность или толщину шрифта. Шрифт может быть жирнее или тоньше обычного начертания. В&nbsp;качестве значения можно использовать ключевое слово или число. Самые часто встречающиеся значения:</p><ul><li><p><code>400</code> или <code>normal</code>&nbsp;—&nbsp;обычный шрифт, значение по&nbsp;умолчанию;</p></li><li><p><code>700</code> или <code>bold</code>&nbsp;—&nbsp;жирный шрифт.</p></li></ul><p>Например:</p><pre><code>h1 {
  font-weight: 400; /* то же самое что и normal */
}

p {
font-weight: bold; /* то же самое что и 700 */
}</code></pre><h2>Свойство text-align</h2><p>Описывает, как выравнивается текст и&nbsp;другие <code>инлайновые</code>
элементы (изображения, инлайн-блоки, инлайн-таблицы и&nbsp;другие) внутри блока по&nbsp;горизонтали.</p><p>Свойство
может принимать следующие значения:</p><ol><li><code>left</code>&nbsp;—&nbsp;выравнивание по&nbsp;левому краю блока, это
значение по&nbsp;умолчанию;</li><li><code>right</code>&nbsp;—&nbsp;по&nbsp;правому краю блока;</li><li><code>
center</code>&nbsp;—&nbsp;по&nbsp;центру блока;</li><li><code>justify</code>&nbsp;—&nbsp;по&nbsp;ширине блока, при этом
слова в&nbsp;строке будут размещаться так, чтобы занять равномерно всё пространство строки (пробелы между словами
в&nbsp;таком случае становятся неравномерными, так как браузер «растягивает» слова в&nbsp;строке).</li></ol><p>Важно
помнить, что свойство <code>text-align</code> применяется именно к&nbsp;самому блоку-контейнеру, внутри которого
находится текстовый контент:</p><pre><code>HTML:
&lt;p&gt;
Я текст внутри абзаца
&lt;/p&gt;

CSS:
p {
text-align: center;
}</code></pre><h2>Свойство vertical-align</h2><p>Свойством можно выравнивать <i>инлайновые</i> элементы относительно
содержащей его строки. Самый простой пример&nbsp;— выровнять картинку <code>&lt;img&gt;</code> по&nbsp;вертикали
в&nbsp;текстовой строке.</p><p>У&nbsp;свойства <code>vertical-align</code> много значений, но&nbsp;самые часто
используемые:</p><ol><li><code>top</code>&nbsp;—&nbsp;выравнивание по&nbsp;верхнему краю строки;</li><li><code>
middle</code>&nbsp;—&nbsp;по&nbsp;середине строки;</li><li><code>bottom</code>&nbsp;—&nbsp;по&nbsp;нижнему краю
строки;</li><li><code>baseline</code>&nbsp;—&nbsp;по&nbsp;базовой линии строки (значение по&nbsp;умолчанию)
.</li></ol><p>В&nbsp;отличие от&nbsp;<code>text-align</code> свойство <code>vertical-align</code> задаётся самому
элементу, а&nbsp;не&nbsp;содержащему его контейнеру:</p><pre><code>HTML:
&lt;p&gt;
&lt;img src="picture.png" alt="Я картинка"&gt;
Я текст внутри абзаца
&lt;/p&gt;

CSS:
img {
vertical-align: middle;
}</code></pre><h2>Свойство color</h2><p>Цветом текста можно управлять свойством <code>color</code>.</p><p>Цвет может
быть задан в&nbsp;виде ключевого слова (полный список ключевых слов приводится
в&nbsp;<a href="https://www.w3.org/TR/css-color-3/#svg-color" target="_blank" rel="nofollow noopener">спецификации</a>).
Например:</p><pre><code>color: black; /* чёрный цвет */
color: red; /* красный цвет */
color: white; /* белый цвет */</code></pre><p>Ещё один вариант указания цвета&nbsp;—
в&nbsp;виде <a href="https://ru.wikipedia.org/wiki/%D0%A8%D0%B5%D1%81%D1%82%D0%BD%D0%B0%D0%B4%D1%86%D0%B0%D1%82%D0%B5%D1%80%D0%B8%D1%87%D0%BD%D0%B0%D1%8F_%D1%81%D0%B8%D1%81%D1%82%D0%B5%D0%BC%D0%B0_%D1%81%D1%87%D0%B8%D1%81%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F" target="_blank" rel="nofollow noopener">
шестнадцатеричного значения</a>. В&nbsp;этом случае цвет формируется из&nbsp;<i>красной</i>, <i>зелёной</i> и&nbsp;<i>
синей</i> составляющих, заданных в&nbsp;виде шестнадцатеричного числа от&nbsp;<code>00</code> до&nbsp;<code>ff</code>.
Помимо шести, цветовой код может содержать три знака, в&nbsp;этом случае второй символ в&nbsp;цветовых составляющих
дублируется первым:</p><pre><code>color: #000000; /* чёрный цвет */
color: #f00; /* красный цвет, то же что #ff0000 */
color: #fff; /* белый цвет, то же что #ffffff */
</code></pre><p>Если не&nbsp;хочется иметь дело с&nbsp;шестнадцатеричными значениями, можно воспользоваться специальной
функцией <code>rgb</code>, в&nbsp;которой указывается цвет в&nbsp;более привычном десятичном виде в&nbsp;диапазоне
от&nbsp;<code>0</code> до&nbsp;<code>255</code> также в&nbsp;виде трёх цветовых составляющих, перечисленных через
запятую:</p><pre><code>color: rgb(0, 0, 0)       /* чёрный, то же что #000000 */
color: rgb(255, 0, 0)     /* красный, то же что #ff0000 */
color: rgb(255, 255, 255) /* белый, то же что #ffffff */</code></pre><p>У&nbsp;функции <code>rgb</code> есть расширенная
версия&nbsp;— <code>rgba</code>. В&nbsp;этом случае помимо указания цвета последним значением указывается степень
непрозрачности цвета&nbsp;— alpha. Значение может быть от&nbsp;<code>0</code> (полностью прозрачный) до&nbsp;<code>
1</code> (полностью непрозрачный):</p><pre><code>color: rgba(0, 0, 0, 0.5)      /* чёрный, непрозрачный на 50% */
color: rgba(255, 0, 0, 0.3)     /* красный, непрозрачный на 30% */
color: rgba(255, 255, 255, 0.9) /* белый, непрозрачный на 90% */</code></pre><h2>Контраст цвета текста
и&nbsp;фона</h2><p>Фоновое изображение и&nbsp;фоновый цвет блока всегда должен достаточно сильно контрастировать
с&nbsp;цветом текста в&nbsp;этом блоке. Чем больше контраст, тем удобнее этот текст читать в&nbsp;разных условиях
освещённости и&nbsp;на&nbsp;разных устройствах. Поэтому если вы&nbsp;задаёте блоку фоновое изображение, нужно
обязательно дополнительно задавать подходящий фоновый цвет. В&nbsp;этом случае, пока изображение загружается, или
в&nbsp;случае, если оно совсем не&nbsp;загрузится, текст всё равно можно будет прочитать:</p><pre><code>p {
/* идеальный контраст: цвет текста белый, цвет фона — чёрный */
background-color: #000000;
color: #ffffff;
}

span {
/* плохой контраст: цвет текста и фона — серые */
background-color: #cccccc;
color: #dddddd;
}
</code></pre><h2>Свойство white-space, управление пробелами</h2><p>Браузер игнорирует множественные пробелы
и&nbsp;переносы строк в&nbsp;HTML-коде. С&nbsp;помощью свойства <code>white-space</code> можно управлять пробелами
и&nbsp;переносами строк. Свойство принимает значения:</p><ul><li><code>nowrap</code>&nbsp;—&nbsp;схлопывает лишние
пробелы и&nbsp;отображает весь текст одной строкой без переносов;</li><li><code>pre</code>&nbsp;—&nbsp;сохраняет пробелы
и&nbsp;переносы как в&nbsp;исходном коде аналогично тегу <code>&lt;pre&gt;</code>;</li><li><code>pre-wrap</code>
&nbsp;—&nbsp;работает как значение <code>pre</code>, но&nbsp;добавляет автоматические переносы, если текст
не&nbsp;помещается в&nbsp;контейнер;</li><li><code>normal</code>&nbsp;—&nbsp;режим по&nbsp;умолчанию: лишние пробелы
и&nbsp;переносы строк схлопываются, текст переносится, пробелы в&nbsp;конце строк удаляются.</li></ul><h2>Свойство
text-decoration</h2><p>Задаёт дополнительное оформление текста. Значения свойства:</p><ol><li><code>underline</code>
&nbsp;—&nbsp;подчёркивание;</li><li><code>line-through</code>&nbsp;—&nbsp;зачёркивание;</li><li><code>overline</code>
&nbsp;—&nbsp;надчёркивание;</li><li><code>none</code>&nbsp;—&nbsp;убирает вышеперечисленные эффекты.</li></ol><p>
К&nbsp;тексту можно одновременно применить несколько эффектов, если перечислить значения через пробел:</p><pre><code>p {
text-decoration: underline; /* подчёркнутый текст */
}

span {
/* подчёркнутый и зачёркнутый текст */
text-decoration: underline line-through;
}</code></pre><p>Свойство <code>text-decoration</code>&nbsp;— <a href="/courses/307/run/7">составное</a>. Оно
раскладывается на&nbsp;отдельные свойства:</p><ul><li><code>text-decoration-line</code>&nbsp;— вид линии: зачёркивание,
подчёркивание или надчёркивание;</li><li><code>text-decoration-style</code>&nbsp;— стиль линии, может принимать
значения:
<ul><li><code>solid</code>&nbsp;— сплошная линия;</li><li><code>double</code>&nbsp;— двойная линия;</li><li><code>dotted</code>&nbsp;— точечная линия;</li><li><code>dashed</code>&nbsp;— пунктирная линия;</li><li><code>wavy</code>&nbsp;— волнистая линия.</li></ul></li><li><code>text-decoration-color</code>&nbsp;— цвет линии.</li></ul><h2>Свойство font-style</h2><p>Свойством можно задать начертание текста. Его основные значения:</p><ol><li><code>normal</code>&nbsp;—&nbsp;обычное начертание;</li><li><code>italic</code>&nbsp;—&nbsp;курсивное начертание;</li><li><code>oblique</code>&nbsp;—&nbsp;наклонное начертание.</li></ol><p>Если задано значение <code>italic</code>, браузер будет пытаться найти в&nbsp;заданном шрифте отдельное курсивное начертание символов. В&nbsp;некоторых шрифтах отдельный курсив предусмотрен.</p><p>Если отдельного курсивного начертания в&nbsp;шрифте не&nbsp;предусмотрено, то&nbsp;браузер сделает текст наклонным, то&nbsp;есть сымитирует курсив. Что равноценно заданию тексту значения <code>font-style: oblique</code>.</p><h2>Свойство text-transform</h2><p>С&nbsp;его помощью можно управлять регистром символов: делать буквы строчными (маленькими) или заглавными (большими). Значения свойства:</p><ol><li><code>lowercase</code>&nbsp;—&nbsp;все строчные;</li><li><code>uppercase</code>&nbsp;—&nbsp;все заглавные;</li><li><code>capitalize</code>&nbsp;—&nbsp;каждое слово начинается с&nbsp;большой буквы;</li><li><code>none</code>&nbsp;—&nbsp;отменяет изменение регистра.</li></ol><h2>Отступы</h2><p>Важный фактор того, что текст в&nbsp;блоке будет удобочитаемым, это наличие свободного пространства в&nbsp;блоке для этого текста. Вокруг текста должно быть достаточно «воздуха», он&nbsp;не&nbsp;должен «прилипать» к&nbsp;краям, ему не&nbsp;должно быть «тесно».</p><p>За&nbsp;отступы в&nbsp;CSS отвечают два свойства: <code>padding</code> задаёт внутренние отступы в&nbsp;блоке, а&nbsp;<code>margin</code> задаёт внешние отступы.</p>
